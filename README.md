<h2> Project Overview </h2>
<p>This project showcases my <b>Personal Portfolio Website</b> designed to present my skills, projects, and contact information in a clean and interactive way. It is built using modern <b>Web Technologies such as HTML, CSS, and JavaScript</b> with a focus on responsive design and smooth user experience.</p>

<h2>Features:</h2>
<ul>
  <li><b>Responsive Design:</b> Works on mobile, tablet, and desktop devices</li> 
  <li><b>Interactive UI:</b> Smooth scrolling and modern layout</li>  
  <li><b>Projects Showcase:</b> Displays my real-time projects</li>  
  <li><b>Clean Design:</b> Minimal and user-friendly interface</li> 
  <li><b>Contact Section:</b> Easy way to connect</li>
</ul>
